The rapid advancement of Large Language Models (LLMs) has pioneered Automated Algorithm Design (AAD)—an emerging paradigm that shifts optimization metaheuristics from laborious human derivation to the autonomous synthesis and evolutionary refinement of complete executable source code structures. 

% PART 2: THE SCIENTIFIC PROBLEM & LANDSCAPE DECEPTION (2 sentences)
% CRITICAL COVERAGE: Mathematically isolate why continuous black-box optimization 
% fails here. Explicitly name how stochastic noise distorts fitness landscapes, 
% introduces false local optima, and causes premature convergence due to deceptive gradients.
This thesis targets a notoriously difficult computational bottleneck within this paradigm: continuous black-box optimization operating within highly non-deterministic, noisy environments. Traditional human-engineered heuristics and existing deterministic code-generation models fundamentally degrade when traversing landscapes corrupted by stochastic sensor or thermal interference; the resulting fitness anomalies introduce deceptive local optima and false gradients that trap search trajectories, forcing premature convergence.

To overcome this vulnerability, this research extends the Large Language Model Evolutionary Algorithm (LLaMEA) framework by constructing a closed-loop evolutionary pipeline running inside a secure, stochastic execution sandbox. Evolving programmatic Python classes are systematically evaluated against selected landscapes from the continuous Black-Box Optimization Benchmark (BBOB) suite wrapped in a dynamic, parameter-controlled noise layer, forcing the language model to rely on reflexive feedback loops and runtime error tracebacks to iteratively engineer alternative logic.
