\chapter{Methodology}\label{chap:methodology}

This chapter details the architectural synthesis, software orchestration, and mathematical formulation of the non-deterministic automated algorithm design pipeline implemented in this thesis. To isolate the direct impact of stochastic pressure on language model-driven heuristic synthesis, our framework tightly couples the generation capabilities of foundational language models with a rigid, noise-injected execution sandbox. The system operates as a closed-loop evolutionary cycle designed to transition Python source-code structures from deterministic assumptions to noise-resilient behaviors. The subsequent methodology outlines our environmental wrapper implementation, the multi-tiered assessment strategy designed to mitigate runaway API overhead, the database tracking layers used to capture structural code mutations, and the formal runtime security containment protocols.
